## 초아봇 명령어 도움말

초아봇을 최초 초대시 접두사(명령어 앞에 붙는 글자)가 `ㅊ`으로 설정됩니다
초아봇의 명령어에는 항상 접두사가 붙습니다 `ㅊ명령어`형태
모든 가이드는 **기본접두사(ㅊ)** 를 기준으로 작성되었습니다

## 초아봇 내부 권한체계 관련
초아봇은 이용자별로 내부 권한체계를 가지고 있습니다

**ban**
>팀파스텔로부터 이용제제를 받은 사용자

**none** 
>등록되지 않은 사용자

**user**
>등록된 일반 사용자

**teampastel**
>봇 관리권한을 가진 관리자
